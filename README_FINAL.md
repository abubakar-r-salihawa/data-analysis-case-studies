# Data Analysis Case Studies

A portfolio collection by **Abubakar Rabiu Salihawa** demonstrating practical data cleaning, exploratory analysis, predictive modelling and clear communication of results.

## Projects

### [Car Price Analysis](car-price-analysis/)

Analyses 201 vehicle records and compares linear, ridge, polynomial and random-forest regression models. The best model, Random Forest, achieved **0.936 test R²** and approximately **2,799.90 RMSE**.

### [Laptop Price Analysis](laptop-price-analysis/)

Analyses 238 laptop records and compares linear, ridge, random-forest and gradient-boosting models. Gradient Boosting achieved **0.477 test R²** and approximately **347.12 RMSE**.

## Skills demonstrated

Python, pandas, NumPy, data validation, missing-value handling, exploratory analysis, categorical encoding, regression modelling, and evaluation with MAE, RMSE and R².
