# Laptop Price Analysis

A reproducible analysis of laptop price drivers using data cleaning, exploratory analysis, categorical encoding and regression modelling.

## Highlights

- 238 laptop records
- RAM is the strongest numeric correlate of price
- Best test model: gradient boosting
- Test R²: approximately 0.48
- Test RMSE: approximately 347 price units

Open `laptop_price_analysis.ipynb` for the interactive analysis or `laptop_price_analysis.html` for a browser-friendly report.
