# Laptop Price Analysis

A reproducible Python case study exploring the hardware and manufacturer characteristics associated with laptop prices.

## Highlights

- 238 laptop records analysed
- RAM and processor characteristics were among the strongest predictors
- Linear, ridge, random-forest and gradient-boosting models compared
- Best test R²: approximately 0.48
- Best test RMSE: approximately 347 price units

The moderate score shows that additional variables such as device age, processor generation, condition and warranty would be needed for a production pricing model.
